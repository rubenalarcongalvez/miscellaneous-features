Resize the window

You can edit the values of each box

Normal width and height

  • width: px
  • height: px

Viewport main sizes

You can set the percentage of screen width or height you want
  • width: vw or vi
  • height: vh or vb

Viewport min size

You can set the percentage of screen min-width or min-height you want
  • width: px
  • min-width: vmin

Viewport max size

You can set the percentage of screen max-width or max-height you want
  • width: px
  • max-width: vmax

Viewport variable sizes (applicable to all previous viewports)

ATTENTION: This can only be proven in native mobile phone or tablet due to its UI, this is done to mobile devices
  • height: svh
  • width: px
Small (i.e. svh) will NOT take in count the dimensions of the interface of the browser (so only the visible web)
  • height: lvh
  • width: px
Large (i.e. lvh) will take in count the dimensions of the interface of the browser (so all the window)
  • height: dvh
  • width: px
Dynamic (i.e. dvh) will adapt to the changes, even if scroll
